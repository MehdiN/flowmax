# Flow Matching

It is a port of the flow matching library found at `https://github.com/facebookresearch/flow_matching` to Jax.

Dependencies:
- `python >= 3.12`
- `jax == 0.9.2`
- `equinox` for the neural network library
- `diffrax` for ODE solver
- `optax` for optimization
- `jaxtyping`
